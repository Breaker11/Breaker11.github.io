hexo框架搭建博客及部署到github

发表于2021-10-29|更新于2021-11-03|博客
|字数总计:641|阅读时长:2分钟|阅读量:

使用hexo博客搭建十分简单且强大，本站也是使用hexo框架进行搭建！

一、首先使用hexo必须要先安装node.js以及git工具

node.js官网 git官网

根据提示一步步进行安装即可。安装完成后可打开命令窗口输入命令查看是否安装成功 node -v

二、下载hexo框架

打开命令行输入命令使用npm或cnpm进行下载：

三、创建个人博客根目录

在本地创建文件夹，接下来所有文件都将保存在文件夹中，所以搞砸了，不用怕直接删掉文件夹重来哈哈。

创建完成文件夹后，进入文件夹，然后右击通过Git Base Here

打开进入目录输入hexo init进行文件初始化。接下来文件夹中会出现一些文件，这些都是自动生成的基础文件。

然后就可以通过 hexo s命令进行启动

可以看到启动http://localhost:4000 端口，这时你可以通过浏览器对端口进行访问预览。你就可以看到hexo基础页面已经完成，并且自动创建一片Hello World文章。

四、创建个人博客github仓库

首先进入github，如果没有注册一个账号。

然后new一个项目，这里要注意==用户部署个人博客的github仓库命名必须符合特定的要求==

格式：==用户名.github.io==

我这里提示是因为我已经有了个人博客仓库。创建完成后里面没有内容,地址部署推送时需要使用:

五、部署上传github

这里我们还需要安装一个插件hexo-deployer-git,部署所需要使用。

进入博客文件夹根目录打开命令行输入:

接下来需要设置文件夹中的==config配置文件，找到最后一行deploy,进行设置，其中repo是github仓库的地址==，

barnch为文件分支结构写main或master均可。

type设置为git；

设置完成后在打开命令窗口输入hexo d进行部署推送到github中。

最后打开github仓库就可以看到上传的文件以后就可以使用地址对博客网页进行访问

https://xxx.github.io

然后就大功告成了，样式和一些其他配置以后都可以自己进行设置。
